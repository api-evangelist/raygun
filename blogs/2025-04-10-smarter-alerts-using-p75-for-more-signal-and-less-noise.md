We’ve rolled out a new feature in Raygun Alerting that gives you more control over how you track and respond to performance regressions. Starting today, you can now use the 75th percentile (P75) as a filter option for page performance data in Real User Monitoring, such as Core Web Vitals and page load time, right alongside the default ‘Average’.
